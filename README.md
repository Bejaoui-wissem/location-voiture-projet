# Projet Location de Voiture - ENICAR
Système de gestion de location de véhicules réalisé en HTML, CSS (Bootstrap) et JS.

## 👥 Équipe de développement (Simulation de branches)
- **bdd-sql** : Structure de la base de données MySQL.
- **ui-front** : Catalogue des véhicules et interface utilisateur.
- **auth-backend** : Système de connexion et sécurité.
- **admin-panel** : Tableau de bord de gestion des véhicules.
- **systeme-resa** : Gestion des réservations clients.

## 🚀 Technologies utilisées
- HTML5 / CSS3 (Bootstrap 5)
- JavaScript (Logique de simulation)
- SQL (Modèle de données)
- Git/GitHub (Workflow par branches)
